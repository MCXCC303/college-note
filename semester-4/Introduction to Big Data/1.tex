\section{概论}%
\label{sec:概论}
信息化浪潮：
\begin{itemize}
    \item 第一次：个人计算机出现
    \item 第二次：互联网出现
    \item 第三次：物联网、云计算、大数据
\end{itemize}
三次信息化浪潮的解决问题：
\begin{itemize}
    \item 第一次：数字化
    \item 第二次：网络化
    \item 第三次：智能化
\end{itemize}

    摩尔定律：由因特尔公司创始人提出；每隔十八个月到两年的时间，CPU的晶体管数量翻一倍，价格降低二分之一
\end{notation}
