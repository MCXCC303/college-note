\begin{notation}
    奇异值分解出来的部分：
    \begin{itemize}
        \item $\bm{U}$ ：$m\times m$ 的方阵，为正交矩阵，其列向量为\textbf{左奇异向量}
        \item $\bm{D}$ ：$m\times n$ 的矩阵，为对角矩阵，其对角线值为\textbf{奇异值}
        \item $\bm{V}$ ：$n\times n$ 的方阵，为正交矩阵，其列向量为\textbf{右奇异向量}
    \end{itemize}
\end{notation}
